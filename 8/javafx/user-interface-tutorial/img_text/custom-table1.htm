This illustration shows a table. The table has three columns: First Name, Last Name, and Email. The Address Book caption is located above the table.

The First Name column contains the following entries: Jacob, Isabella, Ethan, Emma, and Michael. The Last Name column shows the following last names: Smith, Johnson, Williams, Jones, and Brown. The Email column contains the email addresses of the contacts. The email addresses are constructed from the first and last names, and the example.com domain name. For example, the first row, from left to right, contains Jacob, Smith, and jacob.smith@example.com.

The empty rows in the table are painted in light yellow.
