This illustration shows the Custom Colors dialog window. It has the following elements (from left to right and from top to bottom):

  • The color area that displays a range of green colors from the light green to the black.

  • The color bar is a vertical color spectrum. Green is selected.

  • The horizontal color pattern strip with orange as the Current Color and green as the New Color.

  • The settings area. This area consists of three tabs: HSB, RGB, and Web. The HSB tab contains the Hue, Saturation, Brightness, and Alpha sliders. The text fields show the corresponding values as follows: 142 for hue, 22 for saturation, and 77 for brightness.

  • The Opacity slider showing the value of 100.

  • The Save, Use, and Cancel buttons at the bottom right of the Custom Color dialog window.
