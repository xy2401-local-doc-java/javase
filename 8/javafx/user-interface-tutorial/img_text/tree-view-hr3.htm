This illustration shows a screen capture of the Tree View Sample. The application window contains a three-level hierarchical structure. The highest item in the hierarchy is MyCompany Human Resources. This element is accompanied by an icon of a person holding a book. The MyCompany Human Resources root item contains the following tree items: Accounts Department, Sales Department, and IT Support. Each department item is accompanied by the icon of an open folder and a person.

The Add Employee context menu is superimposed on the Accounts Department tree item.
