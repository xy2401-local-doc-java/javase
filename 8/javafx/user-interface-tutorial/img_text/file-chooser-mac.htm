This illustration shows a FileChooserSample running in Mac OS.

The left part is the directory structure that contains the visible following items (from top to bottom): Applications, Desktop, Documents, Downloads, Shared, Devices, and Media. The Devices item is expanded to show Macintosh HD2 and MACINTOSH H.

The right part of the window is the table with the following columns: Name and Date Modified. The table contains the following items: .DS_Store with the modification date Dec 4, 2012 and .localized with the modification date Dec 7, 2011. The two with the .localized item is highlighted with blue.

Below the table are two buttons: Cancel and Open. The Open button is highlighted with blue.
