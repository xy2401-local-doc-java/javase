This illustration shows a View Pictures window. The window is divided into two parts.

The left part is the directory structure, which contains the following items: Contacts, Desktop, Downloads, Favorites, Links, My Documents, My Music, My Pictures, My Videos, Saved Games, Searches, Computer, and System (C:).

The right part of the window contains three rows of folder icons. The first (top) row has the following icons: Contacts, Desktop, Downloads, and Favorites. The second row has the following icons: Link, My Documents, My Music, and MyPictures. The third row contains the following icons: My Videos, Saved Games, and Searches.

Under the directory structure and the pictures is an editable combo box, which is labeled "File name".

Two buttons are located under the combo box. From left to right they are Open and Cancel. The Open button is highlighted with a blue border.
