This illustration shows an Open Resource File window. The window is divided into two parts.

The left part is the directory structure, which contains several items. The root item, Desktop, is expanded to Libraries and Computer. The Libraries directory is selected and it has the following subdirectories (from top to bottom): Documents, Music, Pictures, Subversion, and Videos.

The right part of the window is titled Libraries. The subtitle states the following: "Open a library to see your files and arrange them by folder, date, and other... ."

Under the titles are the following libraries (from top to bottom): Documents, Music, Pictures, Subversion. The Documents library is highlighted.

Under the directory structure and the pictures is an editable combo box, which is labeled "File name".

Two buttons are located under the combo box. From left to right they are Open and Cancel. The Open button is highlighted with a blue border.
