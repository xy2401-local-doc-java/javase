This illustration shows the three color setting panes of the Custom Colors dialog window. The top pane is the HSB pane. It contains the Hue, Saturation, and Brightness sliders. The corresponding text fields display the following values: 171 for hue, 22 for saturation, and 77 for brightness.

The RGB pane, which is below the HSB pane, has the following sliders: Red, Green, and Blue. The corresponding text fields display the following values: 153 for red, 196 for green, and 190 for blue.

At the bottom of the illustration is the Web pane. It contains the Web text field with the following value: #99c4bd.
