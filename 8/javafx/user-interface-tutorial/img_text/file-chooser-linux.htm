This illustration shows a FileChooserSample running in Linux.

The left part of the window is the directory structure, titled Paces. It contains the following components (from top to bottom): Search, Recently Used, fx, Desktop, File System, Documents, Music, Pictures, Videos, and Downloads.

The right part of the window contains the table with the following columns: Name, Size, and Modified. The Name column lists the following items: build, dist, nbproject, src, build.xml, and manifest.mf. The Size column shows that build.xml is of 3.6 KB and manifest.mf is of 82 bytes. The Modified column shows 18:48 for all items. The highest row of the table is highlighted with orange.

Below the table are two buttons: Cancel and Open. The Open button is highlighted with orange.
