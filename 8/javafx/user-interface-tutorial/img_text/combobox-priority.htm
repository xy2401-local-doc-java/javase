This illustration shows a screen capture of the ComboBoxSample application. Its user interface contains several UI controls arranged in rows.

The first row contains two combo boxes indicated by the labels To and Priority. The Priority combo box is expanded. It lists the following items: Highest, High, Normal, Low, and Lowest. The High and Highest items are red, the Low and Lowest items are green, and the Normal item is black.

The second row contains the Subject field with the corresponding label.

The third row is the text area for the message body.

A Send button is at the bottom of the screen.
