The Login Form has two columns and three rows. Numbering for the columns and rows starts at 0. The Welcome text is in column 0, row 0 and spans two columns. The User Name label is in column 0, row 1 and the text field is in column 1, row 1. The password label is in column 0, row 2 and the password field is in column 1, row 2. There is a 10-pixel horizontal gap between the rows and a 10-pixel vertical gap between the columns.
