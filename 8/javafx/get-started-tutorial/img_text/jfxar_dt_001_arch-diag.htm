The JavaFX architecture diagram shows the stack. The top layer, shown in a rounded maroon box, represents the JavaFX public APIs and the JavaFX scene graph. The next layer is the Quantum Toolkit in a rounded blue box. The third layer has the graphics pipeline called Prism in a rounded blue box, Glass Windowing Toolkit (Glass) in a square pea green box, media engine in a square bright green box, and web engine in a square brown box. The fourth layer consists of the Java 2D, OpenGL, and Direct3D (D3D) in rounded blue boxes. The Quantum Toolkit, Prism, Java 2D, OpenGL, and D3D, all shown in rounded blue boxes, make up the JavaFX graphics system. The lowest layer, shown in light purple, is the Java Virtual Machine. The Java Virtual Machine sits underneath the JavaFX graphics system, Glass Windowing Toolkit, media engine, and the web engine.
