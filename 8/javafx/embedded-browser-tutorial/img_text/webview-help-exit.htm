This illustration shows an application with an embedded browser and a row of links.

The browser renders the Help page. The show/Hide Help Topics list is expanded.

Products - Extensive overview of Oracle hardware and software products, and summary of Oracle consulting, support, and education services.

Blogs - Oracle blogging community

Documentation -Landing page to start learning Java. The page contains links to the Java tutorials, developer guides, and API documentation.

Partners - Oracle partner solutions and programs. Popular resources and membership opportunities.

The Exit the Application link is under the list of the help topics.

The row of links is underneath the browser. From left to right the links are: Products, Blogs, Documentation, Partners, and Help. The links are accompanied with icons. The Products icon shows an application window, the Blogs icon shows a group of schematic images of a person, the Documentation icon shows a person with a book, the Partners icon depicts an open box, and the Help icon is a question mark.
