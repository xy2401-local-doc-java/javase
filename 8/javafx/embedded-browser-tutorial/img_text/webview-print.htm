This illustration shows an application with an embedded browser, a row of links, and a combo box.

The browser renders the Java SE Technical Documentation page.

The combo box and the row of links are underneath the browser.

From left to right the links are: Products, Blogs, Documentation, Partners, and Help. Next to the links are icons. The Products icon is represented by an application window. The Blogs icon is represented by a group of schematic images of a man. The Documentation icon is represented by a man with a book. The Partners icon is represented by an open box. The Help icon is a question mark.

The Toggle Previous Docs button is aligned to the right of the links.

There is a context menu beneath the question mark. The Print menu is highlighted, and the mouse cursor is pointing to the menu.
