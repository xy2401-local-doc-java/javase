This illustration shows an application with an embedded browser and a row of hyperlinks.

The browser renders the Products page.

The row of links is underneath the browser. From left to right the hyperlinks are: Products, Blogs, Documentation, Partners, and Help. Next to the links are icons. The Products icon is represented by an application window, the Blogs icon is represented by a group of schematic images of a man, the Documentation icon is represented by a man with a book, the Partners icon is represented by an open box, and the Help icon is a question mark.

The popup window in the middle of the browser shows three options: Open line, Open link in new window, Copy link to clipboard. The Open link in New Window option is highlighted.
