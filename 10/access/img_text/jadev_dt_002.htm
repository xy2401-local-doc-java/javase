This diagram describes how Java Access Bridge and Java Accessibility Utilities components interact with each other.

This diagram represents components and subcomponents with labeled boxes. It consists of three main components:

  • Java Runtime

  • Native Assistive Technology

  • Java Access Bridge for Windows

The boxes Java Runtime and Java Access Bridge for Windows overlap each other. Any subcomponent that is located where these two main components overlap each other belong to both of these components.

The Java Runtime component contains these components:

  • Java Application, which contains the component UI Toolkit, which contains the component Java Accessibility API (JAAPI)

  • Java Accessibility Utilities

The Java Access Bridge for Windows component contains these components:

  • Runtime DLL

  • Windows DLL
