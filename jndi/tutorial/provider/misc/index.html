Miscellaneous

This lesson shows you how to bundle all of the components of your service provider. It also gives you some tips on how to implement the following advanced features:
  • Referrals
  • Link references
  • Event notification
  • LDAP v3 controls and extensions
Adding support for each of these features requires quite a bit of work. This lesson touches on only some aspects of their implementation. This is because the actual implementations will be closely tied to the underlying service and how you have structured the other components of your service provider.
