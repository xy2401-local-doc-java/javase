Schema

This lesson gives an overview of LDAP schema and describes how it maps to the JNDI. It provides details about the schema content of object class definitions, attribute type definitions, syntax definitions, and matching rule definitions. It also contains two examples that illustrate practical uses of the schema.
Windows Active Directory: Only authenticated users may read the Active Directory schema. See the Security Lesson on information on how to perform authentication.

The Active Directory schema is updated through an internal schema tree. The examples in this lesson that perform updates to the schema will not work against Active Directory.
